# TasksPresets List

List all presets user have access to. You can see details by clicking on it.

> **Note**
> This components show only if any `global.tasks-presets-get` permission is granted to the authed user.

### Default

```html

  <ezr-provider ...>
    <ezr-tasks-presets-table />
  </ezr-provider>
</template>
```
