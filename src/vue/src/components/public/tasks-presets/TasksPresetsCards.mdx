# TasksPresets Cards

List all presets user have access to. You can see details by clicking on a little cog.

> **Note**
> This components show only if any `global.tasks-presets-get` permission is granted to the authed user.

## Usage

### Default

```html
<template>
  <ezr-provider ...>
    <ezr-tasks-presets-table />
  </ezr-provider>
</template>
```
