# History Table

List activity of all task user have access to, entries are re-fetched every 5s by default.

> **Note**
> This components show only if any `namespaces.history-get` permission is granted to the authed user.

## Props

<Controls />

## Usage

```html
<template>
  <ezr-provider ...>
    <ezr-history-table :fetchInterval="5000" />
  </ezr-provider>
</template>
```
