# Task Table

List and manage of all task user have access to. You can see details about a task by clicking on it.

> **Note**
> This components show only if any `namespaces.tasks-get` permission is granted to the authed user.

## Usage

### Default

```html
<template>
  <ezr-provider ...>
    <ezr-task-table />
  </ezr-provider>
</template>
```
