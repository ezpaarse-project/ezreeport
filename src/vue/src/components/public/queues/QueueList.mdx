# Queue List

List and manage all queues. You can see jobs in queue by clicking on it.

> **Note**
> This components show only if `general.queues-get` permission is granted to the authed user.

## Usage

### Default

```html
<template>
  <ezr-provider ...>
    <ezr-queue-list />
  </ezr-provider>
</template>
```
