# Template List

List and manage all templates. You can see details about a template by clicking on it.

> **Note**
> This components show only if `general.templates-get` permission is granted to the authed user.

## Usage

### Default

```html
<template>
  <ezr-provider ...>
    <ezr-template-list />
  </ezr-provider>
</template>
```
