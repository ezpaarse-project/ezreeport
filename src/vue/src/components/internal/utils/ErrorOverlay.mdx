# Error Overlay

Wrapper of `v-overlay`.

**Needs parent to have a `postion: relative`.**

## Usage

```vue
<template>
  <v-app>
    <div style="position: relative">
      <ErrorOverlay v-model="error">
    </div>
  </v-app>
</template>

<script>
export default {
  data: () => ({
    error: 'A mock error occured'
  })
}
</script>
```
